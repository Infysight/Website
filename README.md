# InfySight - Startup Company Website

A modern, responsive website for InfySight, a technology startup focused on innovative digital solutions.

## Features

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling, mobile navigation, form validation
- **Performance Optimized**: Fast loading times and optimized assets
- **Contact Form**: Functional contact form with validation
- **SEO Friendly**: Semantic HTML and proper meta tags

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Company mission, stats, and key features
3. **Services Section**: Detailed overview of services offered
4. **Team Section**: Meet the team members
5. **Contact Section**: Contact information and contact form
6. **Footer**: Links, social media, and additional information

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and animations
- **Font Awesome**: Icons
- **Google Fonts**: Inter font family

## File Structure

```
/

├── js/
```

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. For development, use a local server (e.g., Live Server in VS Code)

## Customization

### Colors
The website uses a gradient color scheme based on:
- Primary: #667eea to #764ba2
- Text: #2d3748 (dark), #718096 (medium), #a0aec0 (light)
- Background: #f7fafc (light), #2d3748 (dark)

### Content
Edit the HTML file to customize:
- Company name and branding
- Service descriptions
- Team member information
- Contact details

### Styling
Modify `css/style.css` to:
- Change colors and typography
- Adjust spacing and layout
- Add new components

## Browser Support

- Chrome (latest)

## Performance Features

- Optimized CSS and JavaScript
- Lazy loading for images
- Smooth animations with CSS transforms
- Debounced scroll events
- Intersection Observer for efficient animations

## Contact Form

The contact form includes:
- Client-side validation
- Visual feedback for user actions
- Responsive design
- Accessibility features

Note: The form currently simulates submission. To make it functional, integrate with a backend service or email service.

## Deployment

To deploy the website:

1. **Static Hosting**: Upload files to any static hosting service (Netlify, Vercel, GitHub Pages)
2. **Domain**: Point your domain to the hosting service
3. **SSL**: Ensure HTTPS is enabled for security
